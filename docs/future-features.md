# Long-term Vision

The goal is not simply to build a website.

The goal is to build a complete platform for managing short-term rentals, capable of growing from two apartments to many more without changing the overall architecture.  


---  
  
# Design Principles

The project should always follow these principles:

- simplicity over complexity

- fast loading

- mobile first

- accessibility

- maintainable code

- reusable components

- elegant and timeless design

- privacy by design

- compliance with Italian regulations
