# Purpose

This document records long-term decisions that define the Milan Red Line project.

Only decisions expected to remain valid over time belong here.

---

# PD-001 — Help First

The project always prioritises helping travellers before promoting accommodation.

Status: Active

---

# PD-002 — Hub → Connector → Specialist

The editorial architecture follows a three-level model.

Status: Active

---

# PD-003 — Reading Experience Framework

All new guides must follow the Reading Experience Framework.

Status: Active

---

# PD-004 — Reusable Components

Editorial UI should be implemented through reusable components whenever practical.

Status: Active

---

# PD-005 — Legacy Migration

Whenever a Legacy guide is modified, the objective is to leave it in Current status before the sprint ends whenever reasonably possible.

Status: Active

---

# PD-006 — Documentation First

Documentation is considered part of every release.

Status: Active

---

# PD-007 — Topical Authority Strategy

Long-term SEO growth is based on topical authority rather than isolated keyword targeting.

The project expands by strengthening coherent content clusters before creating new topics.

The four permanent editorial pillars are:

- Accommodation
- Transport
- Day Trips
- Local Knowledge

Status: Active

---

# PD-008 — Success Metrics Activation

Meaningful project KPIs will be introduced only after reliable analytics data becomes available.

Until then:

- avoid artificial KPI reporting;
- use Search Console and Analytics for observation rather than performance scoring;
- do not include permanent Success Metrics sections in project handovers.

Once sufficient data exists, a dedicated Success Metrics section will become part of the project governance.

Status: Active

---

# Change Policy

Existing decisions should normally remain in place.

If a decision changes:

1. mark the previous decision as superseded;
2. create a new decision;
3. explain the rationale.

Avoid rewriting project history.

---

# Maintenance

Update this document whenever a permanent architectural, editorial or strategic decision is adopted.
