# 07_Content_Standards

> **Module Owner:** Content & SEO
> **Status:** Living document

---

# Purpose

This document defines the permanent editorial standards for every Milan Red Line guide.

The objective is to ensure consistency, quality, readability and long-term SEO value.

---

# Editorial Principles

Every guide must:

- answer one primary search intent;
- help before selling;
- use simple, natural English;
- avoid duplicated content;
- remain evergreen whenever possible.

Quality always has priority over publication speed.

---

# SEO Writing Principles

SEO is the consequence of useful content, not its objective.

Always:

- write for people first;
- optimise structure second;
- use keywords naturally;
- answer the reader's question completely.

Never:

- write for search engines;
- repeat keywords unnaturally;
- create pages only to target a keyword.

---

# Search Intent

Before writing, identify:

1. the reader's question;
2. the guide role (Hub / Connector / Specialist);
3. the appropriate content pillar;
4. the next logical guide.

If these are unclear, postpone publication.

---

# Content Compression Pass

Before publication perform a Compression Pass.

Remove:

- duplicated concepts;
- filler;
- unnecessary adjectives;
- repeated introductions.

The goal is a shorter guide with the same informational value.

---

# Tone of Voice

Preferred tone:

- calm;
- practical;
- trustworthy;
- welcoming;
- local.

Avoid exaggerated or promotional language.

Unsupported claims should never appear.

---

# Calls to Action

CTAs should appear only after useful information.

Preferred examples:

- Contact us for availability.
- Explore Apartment Arco and Apartment Gramsci.
- We'd be happy to help.

CTAs must never interrupt the learning experience.

---

# Local Knowledge

Whenever possible include information that generic travel websites usually omit:

- neighbourhood atmosphere;
- practical transport advice;
- everyday services;
- realistic planning tips.

---

# Relationship with SEO

High rankings should result from:

- topical authority;
- complete answers;
- consistent quality;
- strong Knowledge Network;
- useful internal links.

They should never depend on keyword stuffing or thin pages.

---

Update this document whenever editorial writing standards evolve.
